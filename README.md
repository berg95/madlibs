# madlibs
MadLibs program


1. To run you have to compile first, copy this command:    javac Madlib.java
2. Run this using the command:      java Madlib.java x      where x the index of the madlib

- Underscore codes:
- _V = Verb
- _N = Noun
- _n = Number
- _A = Adjective
- _a = Adverb
- _P = Plural noun
